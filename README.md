This project focuses on predicting housing prices using advanced supervised machine learning techniques.

# 🚀 Why XGBoost?

The project uses:

XGBoost (Extreme Gradient Boosting)

XGBoost improves predictions iteratively:
- one model makes predictions
- another learns previous errors
- each new model improves earlier predictions

This makes it one of the strongest algorithms for:
- regression
- tabular datasets
- housing prediction
- Kaggle competitions
